**SEBELUM KERJA SELALU LAKUKAN GIT PULL**

untuk run frontend: 
1. masuk ke directory fe di terminal
2. instal dependency jika belum
```
npm install
```
3. jalankan dev:
```
npm run dev
```

untuk run backend:
1. masuk ke directory server di terminal lain
2. instal dependency jika belum
```
npm install
```
3. jalankan dev:
```
npm run dev
```
